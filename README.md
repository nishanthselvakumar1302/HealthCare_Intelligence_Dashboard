# Healthcare Disease Intelligence Dashboard

## Project Overview

The **Healthcare Disease Intelligence Dashboard** is an end-to-end data analytics project designed to analyze healthcare disease data and uncover insights about patient demographics, disease severity, treatment costs, and geographic disease distribution.

The project integrates **Python for data cleaning and exploratory analysis, SQL for analytical queries, and Power BI for interactive visualization** to provide a complete healthcare analytics solution.

The goal of this project is to help healthcare stakeholders identify **high-risk patient groups, treatment cost drivers, and regional disease patterns**, enabling better healthcare planning and resource allocation.

---

## Business Problem

Healthcare organizations often struggle to understand how diseases impact different populations, how treatment costs vary, and which regions require additional healthcare resources.

Without proper analytics, healthcare decision-makers cannot easily identify **high-risk patients, high-cost treatments, or disease clusters across regions**.

This project addresses that challenge by building a **data-driven analytics dashboard** that provides insights into healthcare trends and disease patterns.

---

## Project Objectives

The main objectives of this project are:

* Analyze healthcare disease data to identify patterns and trends
* Understand patient demographics and disease prevalence
* Evaluate treatment costs and hospitalization patterns
* Identify high-risk patient groups and mortality trends
* Build an interactive dashboard to support healthcare decision-making

---

## Dataset Description

The dataset contains healthcare patient records including:

* Patient demographics (age, gender, income level)
* Disease name and disease category
* Disease severity levels
* Treatment cost information
* Hospital type
* Insurance status
* Hospitalization duration
* Patient outcomes (recovery or death)

These variables allow comprehensive analysis of **disease risk, healthcare costs, and patient demographics**.

## Key Metrics (KPIs)

The dashboard highlights important healthcare performance indicators:

* **Total Patients** – Total number of diagnosed patients
* **Mortality Rate** – Percentage of patients who died from the disease
* **Average Treatment Cost** – Average cost of treatment per patient
* **Average Length of Stay** – Average number of hospitalization days

These KPIs help evaluate **healthcare burden and treatment efficiency**.

---

## Dashboard Features

The Power BI dashboard includes the following features:

* **Disease Selector** – Filter analytics by disease type
* **Disease Information Panel** – Displays disease image and description
* **KPI Cards** – Key healthcare metrics
* **Bookmark Navigation** – Dynamic dashboard sections
* **Geographic Analysis** – Disease cases by state
* **Risk Analysis** – Mortality and severity insights
* **Cost Analysis** – Treatment cost patterns
* **Patient Demographics** – Gender, age, and income distribution

---

## Dashboard Sections

### Overview

Provides a high-level view of disease patterns including geographic distribution, patient age groups, and severity distribution.

### Risk Analysis

Identifies patient risk factors by analyzing disease severity, mortality rates, and hospitalization patterns.

### Cost Analysis

Examines treatment costs across hospital types, insurance status, and disease severity levels.

### Patient Profile

Explores demographic insights including gender distribution, income levels, and urban vs rural population patterns.

---

## Key Insights

The analysis revealed several important healthcare insights:

* Middle-aged patients (40–59 years) represent the largest share of diagnosed cases.
* Severe and critical disease cases significantly increase mortality rates.
* Treatment costs increase with longer hospitalization duration.
* Disease prevalence varies across geographic regions.
* Insurance coverage helps reduce financial burden for patients.

These insights can help healthcare organizations **improve resource planning and patient care strategies**.
